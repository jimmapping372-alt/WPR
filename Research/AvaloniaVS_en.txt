Avalonia for Visual Studio  

Avalonia is a cross-platform UI framework based on XAML, providing a flexible styling system and supporting a wide range of operating systems such as Windows via .NET Framework and .NET Core, Linux via Xorg, and macOS.

The Avalonia for Visual Studio extension includes project and control templates to help you get started. For a getting started guide, see our documentation.

Legacy version support  
If you are using VS2019, you need to install the extension for older versions.

Please use the GitHub repository for issues and discussions: https://github.com/AvaloniaUI/AvaloniaVS/issues
